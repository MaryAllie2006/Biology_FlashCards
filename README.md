# Biology_FlashCards

This project is a simple flashcard application built with HTML, CSS, and JavaScript. It allows users to study biology questions and answers by flipping through virtual flashcards.

Features:

-Flashcards with questions and answers

-Flip card functionality

-Previous and next buttons to navigate through flashcards

-Progress bar to track progress

-Keyboard navigation (left/right arrow keys to change cards, spacebar to flip)
